### Operation
```
1. priority_queue::empty()	  TC: O(1)          Returns whether the queue is empty.
2. priority_queue::size() 	  TC: O(1)          Returns the size of the queue.
3. priority_queue::top()	    TC: O(1) SC: O(n) Returns a reference to the topmost element of the queue.
4. priority_queue::push() 	  TC: O(log(n))     Adds the element ‘g’ at the end of the queue.
5. priority_queue::pop()	    TC: O(log(n))     Deletes the first element of the queue.
6. priority_queue::swap()	    TC: O(1)          Used to swap the contents of one priority queue with another priority queue of the same type and size.
7. priority_queue::emplace()	TC: O(log(n))      Used to insert a new element into the priority queue container.

8. priority_queue value_type 	 Represents the type of object stored as an element in a priority_queue. It acts as a synonym for the template parameter.
```
